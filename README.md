# My Desktop Assets
This is my desktop assets from elementary os

# Requirements must be installed 
* DE       : xfce
* WM       : xfwm
* theme    : adwaita-dark
* icons    : Oranchelo-Green
* terminal : urxvt
* terminal font : Hack
* font     : Fira Code Light 10
* accesories : plank
